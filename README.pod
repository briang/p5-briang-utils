=head1 BRIANG::Utils

Version 0.012

A collection of small, miscellaneous utilities frequently used by the
author, but that aren't worthy of their own distribution.

=head2 INSTALLATION

To install this module, run the following commands:

    perl Makefile.PL
    make
    make test
    make install

Or, just use your favorite CPAN client

    cpanm BRIANG::Utils

=head2 DEPENDENCIES

There are no dependencies on non-core modules. L<Test::Exception> is
required for testing, however.
